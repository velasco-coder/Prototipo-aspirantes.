# 04 - Modulos Funcionales

## Objetivo

Convertir los diagramas en modulos claros. Cada modulo puede transformarse despues en pantallas, endpoints, tablas y casos de prueba.

## Modulo 1: Registro publico de aspirantes

Pantallas relacionadas:

- Registro Aspirantes
- Registro Aspirante - Estudiante
- Registro Aspirante - Estudiante Fam
- Registro Aspirante - Estudiante Proc
- Aspirante Registrado

Funciones:

- Seleccionar tipo de registro.
- Capturar carrera de interes.
- Capturar datos personales.
- Capturar datos familiares.
- Capturar bachillerato de procedencia.
- Capturar promedio.
- Confirmar registro.
- Enviar correo con datos de acceso.

## Modulo 2: Autenticacion

Pantallas relacionadas:

- Inicio de sesion
- Usuario o contraseña incorrecta
- Cambiar contraseña

Funciones:

- Iniciar sesion.
- Validar usuario y contraseña.
- Mostrar error de autenticacion.
- Cambiar contraseña.
- Separar menus segun rol.

## Modulo 3: Portal del aspirante

Pantallas relacionadas:

- Inicio Aspirante
- aspCompletarInfo
- Aspirante Ficha EXANII
- Aspirante Comprobante
- Aspirante Carrera Seleccionada
- Aspirante Datos Personales

Funciones:

- Mostrar bienvenida y datos basicos.
- Mostrar carrera seleccionada.
- Mostrar guia CENEVAL.
- Generar o imprimir pago CENEVAL.
- Subir comprobante de pago CENEVAL.
- Consultar datos personales y familiares.
- Avisar informacion incompleta.

## Modulo 4: Administracion de aspirantes

Pantallas relacionadas:

- Inicio Responsable
- Ver Aspirantes
- Aspirantes Agregar
- Aspirantes Agregar Good
- Ver Aspirantes Activar
- Ver Aspirantes Desactivar
- Editar Aspirante
- Registro Asp 1/2/3

Funciones:

- Listar aspirantes.
- Filtrar aspirantes.
- Agregar aspirante manualmente.
- Editar informacion.
- Activar/desactivar usuario.
- Consultar fecha de registro.
- Consultar escuela de procedencia.
- Consultar carrera seleccionada.
- Consultar ficha de pago CENEVAL.

## Modulo 5: CENEVAL

Pantallas relacionadas:

- Inicio Responsable CENEVAL
- Ver Aspirantes CENEVAL
- Ver Aspirantes CENEVAL Editar
- Resultados CENEVAL

Funciones:

- Subir archivo de resultados.
- Buscar aspirante por matricula/folio.
- Capturar resultado individual.
- Consultar resultado por aspirante.
- Marcar aspirante como evaluado.
- Habilitar aceptacion/no aceptacion.

## Modulo 6: Aceptados y nuevo ingreso

Pantallas relacionadas:

- Inicio Nuevo Ingreso
- Lista Aceptados
- Agregar Nuevo Alumno
- Agregar Nuevo Alumno Good
- Dir Carrera Aceptados
- Dir Academico Aceptados

Funciones:

- Listar aceptados.
- Consultar resultado CENEVAL.
- Consultar estatus de alumno.
- Ver documentos entregados.
- Agregar alumno manualmente.
- Registrar puntaje y formato de resultados.
- Confirmar alta.

## Modulo 7: Documentos de inscripcion

Pantallas relacionadas:

- Subir Documentos
- Subir Documentos Falta
- Subir Documentos Done
- Lista Aceptados Validar Doc
- Lista Aceptados Validar
- Lista Aceptados Validado

Funciones:

- Subir acta de nacimiento.
- Subir certificado de bachillerato.
- Subir comprobante de domicilio.
- Subir ficha de pago de inscripcion.
- Permitir documento adicional.
- Validar documento como valido/no valido.
- Registrar observacion.
- Avisar si faltan documentos.

## Modulo 8: Pago de inscripcion

Pantallas relacionadas:

- Pago Inscripcion
- Solicitar ficha de pago de inscripcion

Funciones:

- Generar referencia de pago.
- Imprimir ficha de pago.
- Consultar estatus de pago.
- Validar/conciliar pago.
- Bloquear alta si el pago no esta validado.

## Modulo 9: Cambio de carrera

Pantallas relacionadas:

- Cambio Carrera
- Cambio Carrera Fin
- Cambio Carrera Motivo
- Cambio Carrera Accept
- Cambio Carrera Rechazar
- Cambio Carrera Change

Funciones:

- Solicitar cambio de carrera.
- Capturar motivo.
- Consultar solicitudes.
- Ver motivo de solicitud.
- Aceptar solicitud.
- Rechazar solicitud con justificacion.
- Actualizar carrera si fue aceptada.

## Modulo 10: Proyeccion de matricula y grupos

Pantallas relacionadas:

- Matricula por carrera
- Proyeccion Matricula
- Proyeccion Matricula A
- Proyeccion Matricula Accept
- Grupos Carrera
- Grupos Carrera Grupo

Funciones:

- Consultar matricula proyectada.
- Definir cantidad total de alumnos.
- Definir cantidad de grupos.
- Definir alumnos por grupo.
- Confirmar proyeccion.
- Consultar grupos por carrera.
- Consultar integrantes de grupo.
